Primero se necesita instalar las diferentes bibliotecas
utilizadas en el software mediante:

python3 install_dependencies.py

Es necesario tener una base de datos en mysql. Hay que rellenar
los datos de conexion de la base de datos en bbdd.py dentro de la función
de conexión y después ejecutar:

python3 recuperarbbdd.py

Para ejecutar el software:

python3 main.py
